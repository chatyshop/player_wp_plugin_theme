Turn WordPress into a self-hosted video platform powered by ChatyPlayer.

== Description ==

CP WP Plugin adds a CP Video post type with video sources, subtitle tracks,
chapters, thumbnail preview sprites, automatic single-video rendering, and the
`[cp_player video="123"]` shortcode.

== Installation ==

1. Upload the `cp-wp-plugin` folder to `/wp-content/plugins/`.
2. Activate CP WP Plugin.
3. Open CP Videos > Add New.
4. Add at least one video source and publish the video.
